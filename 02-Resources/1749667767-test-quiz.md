# Test Quiz
### What is the difference between `DELETE`, `TRUNCATE`, and `DROP`???
- `DELETE`: Removes rows (can be rolled back).  
- `TRUNCATE`: Removes all rows (faster, can't be rolled back).  
- `DROP`: Deletes the entire table.
