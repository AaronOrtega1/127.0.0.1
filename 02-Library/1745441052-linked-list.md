# Linked List
## Summary
- A Linked List is a linear data structure where elements ([[1760038325-node|node]]) are **connected through pointers** rather than stored contiguously in memory. Each node contains a value and a reference to the next node. Linked lists **excel at insertion and deletion** operations (O(1) when you have a reference to the position) compared to arrays which require shifting elements. They're ideal when you need dynamic size, frequent insertions/deletions at known positions, or when memory allocation needs to be flexible. The **tradeoff is O(n) access time for arbitrary elements** versus O(1) for arrays.

### Structure
```java
class Node {
    int value;
    Node next;
}

LinkedList {
    Node head;
    Node tail;
    int length;
}
```
