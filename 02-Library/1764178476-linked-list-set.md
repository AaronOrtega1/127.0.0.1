# Linked List Set
## Problem
Access the [[1760038325-node|node]] in a specific index and update its value.

## Core Idea
- Use [[1764178154-linked-list-get|get(index)]] to find the node
- If it exists, update its value
- Return success/failure for success

## [[1744043004-big-o-timer-and-space-complexity|Time And Space Complexity]]
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
