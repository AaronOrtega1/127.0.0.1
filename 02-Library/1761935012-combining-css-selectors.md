# Combining CSS Selectors
## Summary
- **Group selectors:** can be used to select multiple HTML elements.
```CSS
selector, selector {
  color: blueviolet;
}
```

- **Child:** apply to direct child.
```CSS
selector > selector {
  color: firebrick;
}
```

- **Chaining:** apply where ALL selectors are true.
```CSS
selector.selector {
  color: seagreen;
}
```

- **Combining Combiners:** use multiple selectors.
```CSS
selector selector.selector {
  font-size: 0.5rem;
}
```
